# Capstone Project

## About

This is going to be a description of the application

## Front End

- Written in JavaScript with React

### Requirements

- Node
- NPM

### Usage

- from the root, navigate to the the `ui` directory with:

```code
cd ui
```

- install dependencies with:

```code
npm i
```

- start ui wth:

```code
npm start
```
